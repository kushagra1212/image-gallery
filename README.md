# Image Gallery (React + TypeScript + Vite)

## Description
- Image gallery using React, TypeScript, Vite, and Unsplash API.

## Setup

- Install dependencies 
   ```bash
   npm install
   ```
- Run the development server
   ```bash
   npm run dev
   ```
- Build for production
   ```bash
   npm run build
   ```

## API

- [Unsplash](https://unsplash.com/developers)
